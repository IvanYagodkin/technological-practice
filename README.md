## Технологическая практика

### Описание проекта
Репозиторий содержит решения практических кейсов-задач в рамках технологической практики по направлению «Прикладная информатика». Проект включает разработку программных решений и аналитические материалы.

### Содержание репозитория

* **task1** — решение задачи с одномерным массивом
* **task2** — работа с базовым и производным классами
* **task3** — проектирование базы данных «Туризм»
* **task4** — разработка веб-приложения
* **task5** — аналитический обзор одной из кейс-задач

### Кейс-задачи

#### Задача 1
Работа с массивами. Поиск суммы отрицательных элементов между максимальным и минимальным значениями.

#### Задача 2
Разработка классов. Создание базового и производного классов с демонстрацией их взаимодействия.

#### Задача 3
Проектирование базы данных «Туризм»:
* Создание таблиц
* Определение связей
* Разработка структуры БД

#### Задача 4
Разработка веб-приложения:
* Проектирование архитектуры
* Создание базы данных
* Реализация функционала

#### Задача 5
Аналитический обзор одной из предыдущих задач по критериям:
* Функциональность
* Производительность
* Удобство использования
* Безопасность
* Масштабируемость
* Сопровождаемость
* Переносимость
* Качество кода
* Тестирование
