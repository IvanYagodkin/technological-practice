# Решение задачи по поиску суммы отрицательных элементов

## Описание задачи
Программа находит сумму отрицательных элементов массива, расположенных между максимальным и минимальным значениями.

## Алгоритм решения
1. Поиск максимального и минимального значений в массиве
2. Определение всех позиций максимумов и минимумов
3. Установка границ поиска
4. Подсчет суммы отрицательных чисел в заданном диапазоне

## Пример использования
```python
result = find_sum_between_max_min([3, -1, 5, -3, 2, -4, 5, -2, 1])
print(result)  # Вывод: -7
