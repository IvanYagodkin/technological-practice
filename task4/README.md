# Кейс-задача №4: Разработка Web-приложения

## Описание проекта
Web-приложение, разработанное на Delphi 10.2 с использованием MS SQL Server и IIS для решения бизнес-задач компании.

## Содержание документации

### 1. Обзор проекта
* **Назначение**: создание полноценного Web-приложения
* **Технологии**: Delphi 10.2, MS SQL Server, IIS
* **Цель**: разработка функционального Web-решения

### 2. Структура проекта
```
task4/
├── db/                # Файлы БД
│   └── schema.sql     # SQL-скрипт создания БД
├── web/               # Веб-компоненты
│   ├── units/         # Юнит-файлы Delphi
│   └── forms/         # Формы приложения
├── iis/               # Конфигурация IIS
│   └── config.txt     # Настройки сервера
├── docs/              # Документация
│   └── analysis.md    # Анализ рынка
├── README.md          # Основная документация
└── main.dpr           # Основной файл проекта
```

### 3. Требования к окружению
* **ОС**: Windows Server/Pro
* **Среда разработки**: Delphi 10.2
* **Сервер БД**: MS SQL Server
* **Веб-сервер**: IIS

### 4. Установка и настройка

#### Предварительные требования
* Установленный Delphi 10.2
* Настроенный MS SQL Server
* Конфигурированный IIS

#### Инструкция по установке
1. Клонировать репозиторий
2. Создать БД через schema.sql
3. Настроить подключение в конфигурационных файлах
4. Опубликовать приложение через IIS

### 5. Функционал приложения

#### Основные модули
* **Система авторизации**
* **Управление данными**
* **Админ-панель**
* **Пользовательский интерфейс**
