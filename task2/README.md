# Кейс-задача №2: Демонстрация работы классов

## Описание задачи
Создание программы, демонстрирующей работу методов базового и производного классов

## Структура программы
* Базовый класс (**BaseClass**)
* Производный класс (**DerivedClass**)
* Методы:
  * Базовые методы
  * Переопределяемые методы
  * Собственные методы производного класса

## Пример использования
```python
base = BaseClass(10)
derived = DerivedClass(20, 30)

# Вызов методов
base.base_method()
derived.derived_method()
